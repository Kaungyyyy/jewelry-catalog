# jewelry-catalog(draft v0)
This repository contains a web-first, password-protected jewelry lookbook. The draft is intended to validate catalog structure, and presentation-
not to function like a retail website

## The Stack
Backend: Java + Spring Boot
Frontend: Simple HTML/CSS (lookbook layout)
Auth: Single shared password gate
Data: Static seed data (generic items)
PDF: Manual snapshot export

## Scope (Draft V0- Frozen)
- Category-based browsing
- Static product entries with carat-weight variants and displayed prices
- Shared password access
- PDF snapshot export

## Out of Scope (Draft v1)
- FTP inventory sync
- Live price automation
- More detailed presentations(not focused right now)
